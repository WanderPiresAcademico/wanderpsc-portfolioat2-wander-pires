Requisitos Atendidos

  • Componentização: 10+ componentes React — Header, Hero, SobreMim, DadosPessoais, Formação, Projetos, Contato, Modal, EditBar e DadosContext.
  • Renderização dinâmica: Arrays de objetos (projetos, formações, habilidades, contatos) renderizados via .map().
  • CRUD completo: Adicionar, editar e excluir dados em todas as seções pelo modo edição (Ctrl+Shift+E ou toque triplo no logo).
  • React Context API: Gerenciamento centralizado de estado com DadosContext.
  • Sincronização: Publicação via GitHub API com versionamento para sincronizar entre dispositivos.
  • Responsivo: Layout adaptado para desktop e celular com CSS media queries.
  • Navegação por abas: Cada seção em sua aba respectiva (SPA).
  • Visualizador de código: Botão "Ver Códigos-Fonte" mostra todos os scripts do projeto.
  • Versionamento: Repositório público no GitHub com histórico de commits.
  • Publicação: Projeto publicado na Vercel com deploy automático a cada push.
